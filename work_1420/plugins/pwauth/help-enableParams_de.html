Durch diese Einstellung wird die Athentifizierung via GET Paramtere aktiviert, mit welcher man auf das System zugreifen kann, ohne dass die Standard Authentifizierung durchlaufen werden muss.
Als parameter sind zurzeit verfügbar:
  • als Benutzer: user, username, login
  • als Password: pass, password, passwd, key
Ein Request and den Server würde so aussehen: http://hudson.server.url/jobs/MyJob/?user=MyUser&pass=MyPass.

ACHTUNG: Das Passwort wird bei dieser Methode unverschlüsselt verschickt und kann abgehört werden.
